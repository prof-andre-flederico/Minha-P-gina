1. Fundamentos da Web e HTML5

Período: 20/10/2025 - 24/10/2025

Tema: Fundamentos da Web e HTML5

Conteúdo Programático

  • Arquitetura da Web e protocolos (HTTP, HTTPS, FTP).
  • Estrutura cliente/servidor e funcionamento básico de servidores web.
  • HTML5: estrutura de documento, elementos básicos e semântica.
  • Tags de texto, listas, links, imagens, tabelas e formulários.
Atividade Prática: Construção de página pessoal simples utilizando HTML5.
Referências: SILVA (2011) – Cap. 1 a 4; CARLOS (2014) – Cap. 1 e 2.

Atividade EAD (5h)

Produzir um site estático de duas páginas utilizando apenas HTML5, contendo:

  • Página inicial com título, texto de apresentação, lista de links e imagem.
  • Página de contato com formulário contendo campos de texto, e-mail, telefone e botão de envio (sem integração).

Critérios: Organização do código, uso correto das tags semânticas e estrutura de navegação entre páginas.

Entrega: Upload do projeto em formato ZIP no AVA até 27/10/2025.

2. CSS e JavaScript para Web

Período: 10/11/2025 - 14/11/2025

Tema: CSS e JavaScript para Web

Conteúdo Programático

  • CSS: sintaxe, seletores, cores, fontes, espaçamento, bordas e posicionamento.
  • Layout responsivo (media queries).
  • JavaScript: variáveis, operadores, estruturas de controle e funções.
  • Manipulação do DOM e eventos.
Atividade Prática: Página responsiva com formulário validado em JavaScript.
Referências: BUDD (2008) – Cap. 1 a 5; CASTRO (2014) – Cap. 3 e 4.

Atividade EAD (5h)

Aplicar CSS e JavaScript ao site criado na semana 1, adicionando:

  • Layout responsivo com media queries.
  • Paleta de cores e tipografia personalizada.
  • Efeitos de hover e transições suaves.
  • Validação do formulário com JavaScript (ex.: verificar campos obrigatórios e formato de e-mail).

Critérios: Uso adequado de seletores CSS, responsividade e funcionalidade da validação.

Entrega: Upload do projeto atualizado no AVA até 17/11/2025.

3. Banco de Dados e Programação Cliente/Servidor

Período: 01/12/2025 - 05/12/2025

Tema: Banco de Dados e Programação Cliente/Servidor

Conteúdo Programático

  • Conceitos de banco de dados para web.
  • Modelagem de tabelas e relacionamentos.
  • PHP básico e integração com MySQL.
  • Criação de aplicações dinâmicas (CRUD simples).
  • Boas práticas: segurança, usabilidade e manutenção.
Atividade Prática: Sistema de cadastro de usuários com HTML, CSS, JavaScript, PHP e MySQL.
Referências: ZERVAAS (2007) – Cap. 1 a 6; CASTRO (2014) – Cap. 6 e 7.

Atividade EAD (5h)

Transformar o formulário do site em uma aplicação funcional integrando PHP e MySQL:

  • Criar banco de dados com tabela para armazenar as informações do formulário.
  • Configurar conexão com o banco via PHP.
  • Implementar inserção e exibição dos registros cadastrados.

Critérios: Funcionamento do CRUD básico (Create/Read), segurança básica (uso de mysqli_real_escape_string ou PDO) e clareza do código.

Entrega: Upload do projeto final no AVA até 08/12/2025, com exportação do banco (.sql).
